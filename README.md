# 💬 Connectify

Connectify is a full-stack developer community platform where users can share posts, update profiles, and engage in real-time conversations through group chats.

## 🚀 Features

- 🔐 Secure authentication using JWT and HTTP-only cookies
- 🧑‍💻 User profiles and post sharing system
- 💬 Real-time group chat using Socket.IO
- 🔄 Live message broadcasting and user activity tracking
- 📁 Media upload support (images/files) using Multer + Cloudinary
- ⚡ Smooth and responsive UI built with React
- 🌐 RESTful APIs with Express.js

## 🛠️ Tech Stack

- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Real-time: Socket.IO
- Media: Cloudinary, Multer
- Authentication: JWT, bcrypt

## 📌 Highlights

- Implemented real-time chat with instant message delivery
- Handled user join/leave events and live synchronization
- Built scalable backend with modular API structure
- Ensured secure session management with cookies
